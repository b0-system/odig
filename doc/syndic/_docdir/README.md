RSS and Atom feed parsing

Documentation
=============

You can find it [here](http://cumulus.github.io/Syndic/).

Build Requirements
==================

 * Ptime >= 0.8.0
 * Xmlm >= 1.2.0
 * Uri >= 1.3.1
