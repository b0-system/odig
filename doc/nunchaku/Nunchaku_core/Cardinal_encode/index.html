Encode Cardinality Constraints into Formulas

If we have some constraint max_card ty = 3, we add an axiom exists a b c : ty. forall x:ty. (x = a || x = b || x = c)

Similarly we can encode minimum cardinality max_card ty = 3 into: exists a b c: ty. distinct(a,b,c)
