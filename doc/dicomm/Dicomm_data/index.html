val elements : (int32 * (string * string * vr * vm * bool)) list

elements pair data element tags with their specification (name, keyword, vr, vm, retired) as found in PS 3.6 2011 §6,7,8 and PS 3.7 2011 Annex E.

val element_ranges : (int32 * int32) list

element_ranges pairs a tag t with a mask m. Given a tag t' if t' land m = t then lookup for t in elements for its definition.

val uid_names : (string * string) list

uid_names pairs UIDs and their names as found in PS 3.6 2011 Annex A).
