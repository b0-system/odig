Each side of the connection has its own heartbeat config. It sends a heartbeat every send_every. If it doesn't receive a heartbeat for timeout, it drops the connection. It only checks whether timeout has elapsed when it sends heartbeats, so effectively timeout is rounded up to the nearest multiple of send_every.
