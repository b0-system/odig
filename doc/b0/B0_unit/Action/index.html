val exec : action

exec is an action that B00_std.Os.Exit.exec's a unit's outcome as follows:

  • The executed file is the unit's B0_meta.exe_file.
  • The arguments have B00_std.Fpath.basename of B0_meta.exe_file as the program name and the action's args as arguments.
  • If the unit defines exec_cwd, it is used to determine the cwd otherwise the default of B00_std.Os.Exit.exec is used.
  • If the unit defines exec_env, it is used to determine the environment otherwise the default B00_std.Os.Exit.exec is used.
