Module BatRefList

Reference on lists.

RefList is a extended set of functions that manipulate list references.

Functional Operations
val fold_left : ('a -> 'b -> 'a) -> 'a -> 'b t -> 'a

List.fold_left f a (ref [b0; b1; ...; bn]) is f (... (f (f a b0) b1) ...) bn.

val fold_right : ('a -> 'b -> 'b) -> 'a t -> 'b -> 'b

List.fold_right f (ref [a0; a1; ...; an]) b is f a0 (f a1 (... (f an b) ...)). Tail-recursive.
