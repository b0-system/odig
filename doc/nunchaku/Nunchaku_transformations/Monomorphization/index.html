val monomorphize : ?⁠depth_limit:int -> ?⁠always_mangle:bool -> (term, term) Nunchaku_core.Problem.t -> (term, term) Nunchaku_core.Problem.t * unmangle_state

Filter and specialize definitions of the problem.

First it finds a set of instances for each symbol such that it is sufficient to instantiate the corresponding (partial) definitions of the symbol with those tuples.

Then it specializes relevant definitions with the set of tuples computed earlier.

parameter always_mangle

if true, polymorphic types become new atomic types, e.g. list int becomes list_int (default: false)

parameter depth_limit

recursion limit for specialization of functions

returns

a new list of (monomorphized) statements, and the final state obtained after monomorphization
