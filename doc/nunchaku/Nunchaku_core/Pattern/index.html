Pattern

A pattern is found on the left of a (nested) pattern match. It is built from constructors and variables only.

module TI = TermInner
type id = ID.t
type 'a var = 'a Var.t

type 'a view =
| Builtin of Builtin.t
| App of id * 'a list
| Var of 'a var
