val index_mld : Odig_support.Conf.t -> Odig_support.Pkg.t -> Odig_support.Pkg_info.t -> user_index:string option -> string

index_mld conf pkg pkg_info ~index is an index.mld page for package pkg with info pkg_info. If user_index is Some content that user provided content is integrated in the result (a leading section "{0" is stripped off though) and no module index is generated by the function.

val pkg_list : Odig_support.Conf.t -> index_title:string option -> raw_index_intro:string option -> ocaml_manual_uri:string option -> string

pkg_list c ~index_title ~raw_index_intro ocaml_manual_uri is an HTML page for thepackages that have their documentation generated in c.

  • index_title is a title for the page with the list of packages; if unspecified one is automatically generated.
  • raw_index_intro is the HTML markup that is inserted before the list of packages; if unspecified one is automatically generated.
  • ocaml_manual_uri is a local URI to the OCaml manual if unspecified the online manual is linked.
