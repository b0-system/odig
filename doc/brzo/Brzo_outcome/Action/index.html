Module Brzo_outcome.Action

Predefined actions.

val exec : 'a action

exec _ c _ _ ~artefact executes program artefact using Brzo.Conf.action_args and Brzo.Conf.cwd. Normally the function does not return.

val show_pdf : 'a action

show_pdf _ c _ _ ~artefact shows path artefact in a PDF viewer using Brzo.Conf.pdf_viewer.

val show_uri : 'a action

show_uri _ c _ _ ~artefact shows path artefact in a browser using Brzo.Conf.browser.
