Like map but specialized for tuples with elements of the same type.

mapn f is equivalent to map f f.

val map1 : ('a -> 'c) -> ('a * 'b) -> 'c * 'b

map1 f (x,y) returns (f x,y)

val map2 : ('b -> 'c) -> ('a * 'b) -> 'a * 'c

map2 f (x,y) returns (x,f y)
